This project demonstrates core front-end principles including DOM manipulation, state management, and CSS-based UI/UX design.View Live Demo🚀 FeaturesReal-time Scoring: Dynamic updates for current and total scores.Game Logic: Automated player switching when a '1' is rolled.Responsive Design: Fully playable on desktop, tablet, and mobile devices.Interactive UI: Smooth transitions and visual feedback for active players.🛠️ Tech StackToolUsageHTML5Semantic structureCSS3Layout and stylingJavaScript (ES6+)Game logic and DOM manipulationGitHub PagesDeployment and hosting📸 Preview<p align="center"><img width="80%" alt="Dice Game Preview" src="https://github.com/user-attachments/assets/920434d0-9a7f-4e78-85ad-7f76cb469aa7" /></p>🕹️ How to PlayRoll the Dice: Each roll adds to your "Current" score.Hold: Add your current score to your total score and pass the turn.The Trap: If you roll a 1, your current score is wiped, and it becomes the next player's turn.Winning: The first player to reach the target score (e.g., 100) wins!
